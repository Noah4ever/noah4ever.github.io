# thiering.org

Personal portfolio website built with React, TypeScript, and Vite.

## Build it yourself

### Requirements

- [Node.js](https://nodejs.org/)
- npm

### Setup

```bash
git clone https://github.com/Noah4ever/noah4ever.github.io.git
cd noah4ever.github.io
npm install
```

### Run locally

```bash
npm run dev
```

Open `http://localhost:5173`.

### Build for production

```bash
npm run build
```

## How it works

- React + TypeScript app with Vite
- Main content is in `src/pages` and `src/components`
- Styling is SCSS in `src/styles`
- Text content is in `src/locales`

## Deploy

Push to `main` and GitHub Pages deploys it to [thiering.org](https://thiering.org).
